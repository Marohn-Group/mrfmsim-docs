Component#

An MRFM experiment typically consists of several components, such as a resonator, a magnet, and a sample. These components can be grouped to simplify the simulation input.

mrfmsim.component.ComponentBase()

The base component class defines the __repr__ method.

The ComponentBase class is an abstract class with a default metadata output. The attrs_to_dict method is to output all public attributes.

Units#

To display the units of the parameters, the class attribute _units should be defined when inheriting the class. See units for detailed units definition.
